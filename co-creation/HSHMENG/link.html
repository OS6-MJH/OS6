如何修改 link.json

本页面将教你如何编辑 link.json，从而在目录页面中新增、修改或删除条目。

1. link.json 在哪里？

它通放在 co-creation.html 同级的目录中，例如： 

/co-creation.html
/link.json

2. link.json 的结构是什么？

它是一个由多个对象组成的数组，每个对象代表目录中的一条内容：

{
    "topic": "如何生成自己的链接？",
    "author": "HSHMENG",
    "intro": "教学",
    "link": "HSHMENG/link.html",
    "tag": "官方教学"
}

每个字段的含义如下：

  • topic：标题（显示在卡片顶部）
  • author：作者名称
  • intro：简介（卡片中的描述）
  • link：点击后跳转的链接
  • tag：分类标签（用于筛选）

3. 如何新增一条内容？

只需在数组末尾添加一个新的对象，例如：

[
    {
        "topic": "如何生成自己的链接？",
        "author": "HSHMENG",
        "intro": "教学",
        "link": "HSHMENG/link.html",
        "tag": "官方教学"
    },
    {
        "topic": "这是新增的内容",
        "author": "你的名字",
        "intro": "这里写简介",
        "link": "your-folder/your-page.html",
        "tag": "前端"
    }
]

注意： 每个对象之间要用逗号 , 分隔。

4. 如何修改一条内容？

找到对应的对象，直接修改字段即可。例如修改标题：

"topic": "新的标题"

5. 如何删除一条内容？

删除对应的对象即可，例如删除第二条：

[
    {
        "topic": "如何生成自己的链接？",
        "author": "HSHMENG",
        "intro": "教学",
        "link": "HSHMENG/link.html",
        "tag": "官方教学"
    }
]

6. 保存后刷新页面即可生效

因为目录页面使用 fetch('./link.json') 动态加载数据， 所以只要保存 JSON 文件并刷新浏览器，就能看到最新内容。

7. JSON 编辑常见错误

  • 漏写逗号 ,
  • 字符串没有用双引号包裹
  • 最后一个对象后面多写了逗号
  • JSON 不是 JS，不能写注释
